Installation

Pre-built packages for Linux, macOS, and Windows are attached to every release. Pick the asset for your platform, verify its checksum, and install. There is nothing to configure afterwards — the MCP server starts automatically the first time you open a project.

Pick your download

PlatformAssetNotes
Linux (Debian/Ubuntu)liteskill-vr_<version>_amd64.debAlso installs the headless MCP server.
Linux (Fedora/RHEL)liteskill-vr-<version>-1.x86_64.rpmAlso installs the headless MCP server.
Linux (any distro)liteskill-vr_<version>_amd64.AppImageSelf-contained; no install step.
macOS (Intel)liteskill-vr-x86_64-apple-darwin.app.tar.gz—
macOS (Apple Silicon)liteskill-vr-aarch64-apple-darwin.app.tar.gz—
Windowsliteskill-vr_<version>_x64-setup.exeNSIS installer (recommended).
Windows (MSI)liteskill-vr_<version>_x64_en-US.msiFor managed/enterprise deploys.

Verify your download

Every release ships a SHA256SUMS.txt. Confirm the file you downloaded matches before installing:

# Linux
sha256sum -c SHA256SUMS.txt --ignore-missing

# macOS
shasum -a 256 -c SHA256SUMS.txt --ignore-missing
# Windows (PowerShell) — compare against the matching line in SHA256SUMS.txt
Get-FileHash .\liteskill-vr_<version>_x64-setup.exe -Algorithm SHA256

Install

Linux — Debian / Ubuntu

sudo apt install ./liteskill-vr_<version>_amd64.deb

Installs the desktop app and the headless liteskillvr-mcp binary to /usr/bin/, and registers the .lsvr file type.

Linux — Fedora / RHEL

sudo dnf install ./liteskill-vr-<version>-1.x86_64.rpm

Linux — AppImage

chmod +x liteskill-vr_<version>_amd64.AppImage
./liteskill-vr_<version>_amd64.AppImage

The AppImage is self-contained and needs no installation. If your kernel lacks FUSE, run it with --appimage-extract-and-run.

macOS

tar -xzf liteskill-vr-<arch>-apple-darwin.app.tar.gz
mv "LiteSkill VR.app" /Applications/

The bundle is unsigned, so the first launch needs Gatekeeper approval: right-click the app → Open → Open, or clear the quarantine attribute:

xattr -dr com.apple.quarantine "/Applications/LiteSkill VR.app"

Windows

Run liteskill-vr_<version>_x64-setup.exe and follow the installer, or install the MSI silently for managed deployments:

msiexec /i liteskill-vr_<version>_x64_en-US.msi /qn

First launch

Open LiteSkill VR, then create or open a project file via the standard file dialog. The MCP server starts automatically and its port appears in the status bar — that's what your AI agent connects to. Continue with Your first project.

Headless / server installs

For CI, servers, and SSH sessions where WebKitGTK isn't available, install only the MCP server. The Linux .deb / .rpm already place it at /usr/bin/liteskillvr-mcp; otherwise grab the standalone liteskillvr-mcp-headless-<target-triple> asset from the release. See Headless MCP server.

Updating

Install the newer package over the old one — apt/dnf upgrade in place, the Windows installer replaces the prior version, and for AppImage/macOS you replace the file or bundle. Your .lsvr projects are independent files and are never touched by an upgrade.

Uninstalling

# Debian/Ubuntu
sudo apt remove liteskill-vr

# Fedora/RHEL
sudo dnf remove liteskill-vr

# AppImage / macOS — just delete the file or the .app bundle

On Windows, uninstall from Settings → Apps. Project .lsvr files are left in place; delete them manually if you no longer need them.
