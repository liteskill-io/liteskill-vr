Building From Source

Once the prerequisites are in place, build release artifacts with the Tauri CLI.

Release bundle

# Release binary + installers (.deb / .rpm / .AppImage on Linux)
APPIMAGE_EXTRACT_AND_RUN=1 NO_STRIP=1 pnpm tauri build

Binary only

# Binary only (no .deb/.rpm/.AppImage bundles)
pnpm tauri build --no-bundle

Headless MCP binary

The GUI-less MCP server is a separate Cargo binary target and needs no WebKitGTK/Tauri system dependencies:

cd src-tauri
cargo build --release --bin liteskillvr-mcp --no-default-features

See Headless MCP server for how to run it.
